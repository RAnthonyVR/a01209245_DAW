Preguntas

¿Por qué es una buena práctica separar el modelo del controlador?

Porque ayuda a mantener el código lo más simple posible, es decir, reparte cada parte por responsabilidad, facilitando el proceso de desarrollo y mantenimiento. Además de facilitar la detección de errores y aplicación de prevención de ataques

¿Qué es SQL Injection?

Es una vulnerabilidad en la que el atacante envía instrucciones SQL por medio de las entradas o formas de la página para que entre dentro de un código SQL con fines de manipular una base de datos. Y en consecuencia administrar la base de datos. [2]

¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection?

Para evitar este tipo de ataque podemos parametrizar las consultas de SQL, es decir, usar tipos de texto y otros formatos para las entradas de información. No dar mucha información sobre el error. Y no aceptar caracteres sospechosos u especiales. [3]
