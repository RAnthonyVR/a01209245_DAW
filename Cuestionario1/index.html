  • ¿Qué es entonces una base de datos - DB?
    Es un conjunto de información o datos que tienen una estructura organizada de tal forma que una computadora pueda acceder a la información solicitada para el análisis o toma de decisiones principalmente. Puede definirse también como una colección de registros, que contiene a su vez campos en una entidad, tal como una organización o persona. [1]
    Existen distintos tipos de modelos de bases de datos tales como: modelo plano, jerárquico, de red y relacionales. Una base de datos generalmente cuenta con un modelo o esquema que describe las entidades de la componen y las relaciones de estas. [2] Los componentes de una base de datos son los siguientes:
    1. Dato.
    2. Relaciones.
    3. Constricciones.
    4. Esquemas. [4]
  • ¿En qué casos es conveniente usar bases de datos?
    Las bases de datos se utilizan casi, en todo lugar, especialmente en telecomunicaciones, universidades, agencias de gobierno, corporaciones, por mencionar algunas. [1]
    La motivación para el uso de bases de datos en lugar de archivos se debe a que se utilizan archivos utilizados por distintos usuarios, la integración de los datos con menor redundancia y actualización de transacciones complejas. [3]
  • ¿Qué es un sistema de gestión de base de datos - DBMS?
    Es un sistema de software que tiene como función el facilitar la recuperación y almacenamiento de datos rápidamente y en una estructura adaptada a las necesidades de un usuario. Normalmente estos datos son almacenados de manera no-volátil o persistente, de tal forma que al ser apagado el sistema que los almacena estos no se pierden. Aunque también se utiliza la memoria volátil o RAM porque el acceso a este es más rápido. Sin los sistemas de gestión de bases de datos, sería más difícil dirigir cualquier actividad de negocio. [1]
  • Enlista y explica las funciones/responsabilidades que tienen los DBMS.
    • Almacenamiento persistente: Se utiliza memoria no volátil para el almacenado de datos.
    • Definición de datos: Es información que provee información sobre datos. (Metadato)
    • Manipulación de datos: Peticiones (Query) de lectura, inserción, actualización y eliminación de datos.
    • Acceso muy rápido: Se utilizan índices.
    • Seguridad de datos: Prevé que los usuarios no autorizados puedan tener acceso a los datos.
    • Recuperación de fallas de software y hardware.

    En el proyecto de la materia Organización Computacional, el proyecto consistió el desarrollar un vivario inteligente que monitoreara en tiempo real la humedad, iluminación, temperatura y filmación del vivario para el cuidado de las plantas y los animales presentes en el vivario. La intención era que el usuario pudiese desde cualquier parte obtener información del vivario.
