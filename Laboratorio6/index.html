Preguntas

  • ¿Cuáles son las diferencias entre los posibles valores de la propiedad position?
    La propiedad position es utilizada para posicionar algún elemento en la pagina web.
    • static: Un elemento es asignado este valor por default esta posición. Lo que hace es que hace que el elemento se encuentre estático en la posición donde le corresponde dentro de la página. Las propiedades top, left, right y bottom por lo tanto no tienen efecto en el posicionamiento del elemento.
    • relative: Permite posicionar un elemento respecto a la posición por defecto del elemento dentro de la página. Se pueden utilizar las propiedades top, left, right y bottom para posicionar el elemento dentro de la página. [1]
    • absolute: El elemento se posiciona respecto a su anterior elemento que no es estático.
    • fixed: El elemento se posiciona respecto a la ventana del buscador.
    • fixed: El elemento se posiciona respecto a la ventana del buscador.
    • sticky: El elemento se posiciona respecto al desplazamiento o movimiento del cursor. Se podría decir que es una combinación entre relative y fixed.
    • initial: Pone la propiedad del elemento en su valor por defecto.
    • inherit: Hace que el elemento herede la propiedad que tiene el elemento padre. [2]
  • ¿Cuáles son los valores estándar para la propiedad visibility?
    La propiedad visibility en CSS especifica si un elemento es visible o no. El valor por default es visible, también se puede utilizar el valor hidden para que se oculte, aunque también toma espacio. Valor collapse elimina elementos de una tabla. El valor initial pone la propiedad en su valor por defecto e inherit hace que herede la propiedad de sus padres. [3]
  • ¿Qué es el zIndex y para qué sirve?
    Es un a propiedad que especifica el orden de un elemento. Esto se hace siguiendo un orden de stack. Solo funciona en absolute, relative y en fixed. Se utiliza para que puedas colocar un elemento encima de otro. [4]
