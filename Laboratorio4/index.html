Desciprción del problema:
Two neighboring kingdoms decided to build a wall between them with some gates to enable the citizens to go from one kingdom to another. Each time a citizen passes through a gate, he has to pay one silver coin.
The world can be represented by the first quadrant of a plane and the wall is built along the identity line (i.e. the line with the equation x = y). Any point below the wall belongs to the first kingdom while any point above the wall belongs to the second kingdom. There is a gate at any integer point on the line (i.e. at points (0, 0), (1, 1), (2, 2), ...). The wall and the gates do not belong to any of the kingdoms.
Fafa is at the gate at position (0, 0) and he wants to walk around in the two kingdoms. He knows the sequence S of moves he will do. This sequence is a string where each character represents a move. The two possible moves Fafa will do are 'U' (move one step up, from (x, y) to (x, y + 1)) and 'R' (move one step right, from (x, y) to (x + 1, y)).
Fafa wants to know the number of silver coins he needs to pay to walk around the two kingdoms following the sequence S. Note that if Fafa visits a gate without moving from one kingdom to another, he pays no silver coins. Also assume that he doesn't pay at the gate at point (0, 0), i. e. he is initially on the side he needs.

Input
The first line of the input contains single integer n (1 ≤ n ≤ 105) — the number of moves in the walking sequence. The second line contains a string S of length n consisting of the characters 'U' and 'R' describing the required moves. Fafa will follow the sequence S in order from left to right.

Output
On a single line, print one integer representing the number of silver coins Fafa needs to pay at the gates to follow the sequence S.

Examples:

input
7
URRRUUU
output
2

input
6
RURUUR
output
1

Preguntas

  • ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
    Cabe mencionar JavaScript y Java no están relacionados en sí, JavaScript decidió incluir al nombre el “Java” porque cuando surgió JavaScript, Java era muy popular.
    Diferencias:
    • JavaScript es un lenguaje de buscador basado en objetos. Java es un lenguaje orientado a objetos.
    • Java es compilado para su ejecución mientras que JavaScript es interpretado.
    • Java es más estricto y requiere que las variables sean declaradas y revisadas en tiempo de compilación antes de ser usadas. JavaScript es menos estricto en cuanto a reglas de sintaxis.
    • Java puede ejecutarse en cualquier buscador o maquina virtual, JavaScript solo puede correr en buscador.
    • Los objetos de JavaScript están basados en prototipos mientras que los objetos de Java están basados en clases.
    • JavaScript requiere menos memoria que Java. [1]

    Semejanzas:
    • Ambos pueden trabajar en un servidor o en un buscador.
    • Ambos tienen frameworks y librerías.
    • Comparten una sintaxis similar a C.
    • Son muy utilizadas en aplicaciones web del lado del cliente. [2]
  • ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)
    • getFullYear(): Obtiene el año como número de cuatro digítos.
    • getMillisecods(): Obtiene los milisegundos.
    • getTime(): Obtiene el tiempo.
    • getDay(): Obtiene el dia de la semana en número
    • getDate(): Obtiene el dia del mes en número. [3]
  • ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)
    • concat(): Concatena dos arreglos.
    • every(): Verifica si todos elementos cumplen una condición.
    • find(): Busca un elemento del arreglo que cumpla una condición
    • pop(): Elimina el ultimo elemento de un arreglo y lo regresa.
    • join(): Forma una cadena con los elementos del arreglo.[4]
  • ¿Cómo se declara una variable con alcance local dentro de una función?
    Para declarar una variable local dentro de una función solamente basta con declarar la función e incluir la declaración de la variable dentro de la misma, como se muestra a continuación:
    function myFuncion() {
    var carro = "Prius";
    }

    Esto significa que solamente se podrá acceder a la variable adentro de la función. Este tipo de variables son creadas cuando la función comienza y se elimina cuando la función termina. [5]
  • ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?
    Que las variables globales pertenecen al objeto window y no se eliminarán al terminar la función sino hasta que la ventana del buscador sea cerrada. Es recomendable que esto no se haga a menos que sea con alguna intención justificada específica. [6]
  • ¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares? ¿Para qué podrías utilizar esto en una aplicación web?
    Se puede usar el método match() que busca las coincidencias para una expresión regular y regresa los patrones. Regresa null si no se encontró. Se puede utilizar en una página web para encontrar contenido específico dada una expresión regular, facilita su búsqueda en toda la página.
